Política de Privacidad

Esta es una política de privacidad para una aplicación de escritorio de uso exclusivamente personal.

Almacenamiento de Datos

Los datos de autenticación (tokens de OAuth 2.0) generados por Google se guardan localmente en la máquina del usuario dentro de un archivo llamado token.json.

Esta información no se transmite, comparte, ni almacena en ningún otro lugar fuera del ordenador personal del usuario.

Uso de Datos

La aplicación utiliza estos tokens de autenticación únicamente para acceder a los datos de Google Calendar y Google Tasks del propio usuario, según las solicitudes que el mismo usuario realiza a través de la ejecución del script. Los datos no se utilizan para ningún otro propósito.
